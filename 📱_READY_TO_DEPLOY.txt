🆘 IF ISSUES ARISE
═══════════════════════════════════════════════════════════════════════════════

Problem: Build fails
Solution:
  1. Check Node version: node --version (should be 16+)
  2. Clear cache: rm -rf .next node_modules
  3. Reinstall: npm install
  4. Rebuild: npm run build

Problem: Styles not applying
Solution:
  1. Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
  2. Restart dev server: npm run dev
  3. Check tailwind.config.ts is correct

Problem: Logo not displaying
Solution:
  1. Check OgoulaBrandMark.tsx has SVG code
  2. Verify React import works
  3. Clear Next.js cache: rm -rf .next

Problem: Colors look wrong
Solution:
  1. Check globals.css CSS variables
  2. Verify hex colors match specification
  3. Clear browser cache completely
